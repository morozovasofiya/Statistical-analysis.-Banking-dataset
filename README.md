# Statistical-analysis.-Banking-dataset

Учебный Jupyter Notebook с практическими заданиями по прикладной статистике, анализу данных и проверке гипотез на реальных и синтетических выборках.

## О проекте

В ноутбуке собраны задачи по статистическому анализу данных с использованием Python. Работа включает:

Описательную статистику, визуальный анализ распределений, ядерные оценки плотности, заполнение пропусков, проверку гипотез о нормальности, поиск выбросов, сравнение выборок статистическими критериями, корреляционный анализ, анализ таблиц сопряженности, проверку мультиколлинеарности, дисперсионный анализ, построение регрессионных моделей.

Проект выполнен в формате учебного практикума и демонстрирует применение классических статистических методов на нескольких датасетах.


### Что анализируется

**1. Датасет кредитного скоринга**
— финансовые характеристики клиентов;

— анализ зависимости признаков от кредитного рейтинга;

— визуализация распределений и категориальных зависимостей.

**2. Датасет фильмов**
— пример анализа выборки малого объема;

— проверка гипотез о нормальности на реальных данных.

**3. Датасет шахматных партий**
— сравнение средних;

— применение t-критериев;

— оценка мощности критериев.

## Структура ноутбука

### 1. Аппроксимация распределений
Гистограммы, KDE (ядерные оценки плотности), сравнение распределений по группам

### 2. Графический анализ
cdplot, dotchart, boxplot, stripplot.

### 3. Анализ нормальных выборок
ECDF, QQ-plot, envelope-графики, критерии нормальности:, Колмогорова–Смирнова, Шапиро–Уилка, Андерсона–Дарлинга, Крамера–фон Мизеса, Лиллиефорса, Шапиро–Франсия

### 4. Анализ выбросов
Критерий Граббса, Q-тест Диксона, визуализация выбросов

### 5. Проверка статистических гипотез
Одновыборочный t-критерий Стьюдента, двухвыборочный t-критерий, односторонние и двусторонние проверки, критерий Манна–Уитни, Левене, Бартлетта, Флигнера–Килина

### 6. Корреляционный анализ
Коэффициент Пирсона, коэффициент Спирмена, коэффициент Кендалла

### 7. Анализ категориальных данных
Хи-квадрат, точный тест Фишера, тест МакНемара, критерий Кохрана–Мантеля–Хензеля

### 8. Мультиколлинеарность
Корреляционная матрица, VIF (factor inflation)

### 9. Регрессионные модели
Линейная регрессия, полиномиальная регрессия, оценка качества аппроксимации
